# 🤖 AI Resume Checker Bot (No-Code Project)

This project is a no-code AI-powered resume checker built using automation tools.

## 🚀 Tools Used
- Telegram Bot
- Make.com (Automation)
- Google AI Studio (API)

## ⚙️ How It Works
1. User sends resume in Telegram
2. Telegram triggers Make.com scenario
3. Resume is sent to Google AI for analysis
4. AI generates feedback
5. Response is sent back to user
